# data/

이 디렉토리는 런타임 상태를 보관하는 로컬 저장소입니다. **저장소에 커밋되지 않습니다** (`.gitignore` 참조).

- `pages/*.json` — 페이지 상태 (제목, 서비스, 블록 목록)
- `assets/*` — 업로드/AI 생성/원격 갤러리에서 임포트한 이미지 원본
- `assets.json` — 자산 인덱스

Turso libSQL 로 이행하기 전까지 임시 스토리지입니다. `lib/storage.ts` 가 이 디렉토리를 참조합니다.

새로 clone 한 저장소에서는 비어 있어야 정상입니다 — `npm run dev` 로 페이지 만들고 자산 업로드하면 자동 생성됩니다.
